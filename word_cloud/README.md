当一个接口有多个服务提供者时, dubbo支持以下几种策略在服务提供者之间进行负载均衡的转发:
1. 加权随机
2. 最小活跃
3. 一致性hash
4. 加权轮询

对于每个策略都有对应的代码及讲解, 可以作为负载均衡相关知识的学习材料
